# Confetti Spawner

A simple and fun application to spawn confetti on your desktop!

## Features

- Spawns confetti on command.
- Includes different types of confetti patterns.
- Portable execution with `Confetti-Portable`.

## Demo

![Confetti Demo](./demo.gif)

## How to use

1. Download the latest release from the [Releases](https://github.com/JudeCJz/confetti-spawner/releases) page.
2. Run `Confetti.exe` from the ZIP file.
3. Enjoy the show!

## Development

- Language: C#
- Source File: `Confetti.cs`
- Build Environment: Visual Studio / .NET
